# artifacts/

Runnable HTML/JS demonstrations of EDF behaviors. Each artifact composes pattern-makers, worlds, pressures, and readouts to show emergent system behavior.

Files will move into this folder as the user finishes sorting `AppDev/`. Until then, artifacts live in `AppDev/` (deliberate staging area; do not auto-clean).

Naming rule: artifacts describe what they do, not cutesy. The project glossary (when ported) is the canonical source for naming language.
